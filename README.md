# WiFiCall
实现同一局域网设备之间语音通话功能

## 使用方法
1.两台设备都安装此应用，按照提示给对应的权限
2.两台设备连接同一个WiFi，或者其中一台设备开启热点，另外一台设备连接这个热点
3.当两台设备在同一个局域网之后。在手机的WLAN设置中查询自己手机当前的IP地址，然后在应用中设置。
